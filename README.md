<h2>Overview</h2>
Patima is an Android Application for inpainting the Head portion for headless Buddha statues that can be found in Anuradhapura Era. As the Final year project of University (2024), this Application is integrated with few other components as mentioned in the Project's main Github Repository.
<br></br>
<a href="https://github.com/MalinduLiyanage/OnesandZeros_Patima">Visit Main Repository of Patima Project</a>
<br></br>
To access the first prototype of this application, which runs with a local SQLite DB, 
<a href="https://github.com/MalinduLiyanage/Patima_App_Android_Java_YOLOv8_SQLite">Click Here.</a>

<h2>Features</h2>
<ul>
  <li>Realtime object detector based Camera preview, with Torch support, and touch to focus features</li>
  <li>Access to device's Location info</li>
  <li>Inbuilt YOLOv8 based statue object suitability detection. <a href="https://github.com/MalinduLiyanage/Buddha_Statue_Head-Body_Detection_YOLOv8">Click to visit the Repo.</a></li>
  <li>Oracle Cloud based Database with Object Segmentation and GAN Machine Learning Models for inpainting, powered by Django backend framework and Django REST API.</li>
  <li>User Session Management</li>
</ul>
